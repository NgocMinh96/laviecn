### Manual

Copy and paste the following code into your project.

<Steps>
  <Step>
    Copy and paste the following code into your project.

    <ComponentSource name="scroll-to-top" />

  </Step>
  <Step>
    Add the following styles to your `global.css` file.

    ```css
      @theme inline {
        --scroll-top: var(--scroll-top);
        --scroll-top-bg: var(--scroll-top-bg);
        --scroll-top-track: var(--scroll-top-track);
      }

      :root {
        --scroll-top: oklch(0.55 0.01 286);
        --scroll-top-bg: oklch(0.55 0.01 286 / 10%);
        --scroll-top-track: oklch(0.55 0.01 286 / 20%);
      }

      .dark {
        --scroll-top: oklch(0.92 0 286);
        --scroll-top-bg: oklch(0.92 0 286 / 15%);
        --scroll-top-track: oklch(0.92 0 286 / 20%);
      }
    ```

  </Step>
  <Step>

    Update the import paths to match your project setup.

  </Step>
</Steps>
